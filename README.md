# **Анализ продаж сети кофеен**
Владелец небольшой сети кофеен планирует расширить свой бизнес и открыть несколько новых точек в течение ближайших 4 месяцев. В связи с этим он хочет понять, как распределяются продажи по существующей сети кофеен, какие виды продукции вносят наибольший вклад в объем продаж, а от каких, возможно, стоит отказаться, кто является основным клиентом сети. Заказчик предоставил датасет в формате csv с выгрузкой о транзакциях и клиентах из его CRM-системы.

**Моя задача**:

Проанализировать полученные данные и представить аналитический отчет, содержащий выводы и рекомендации для Заказчика, которые помогут ему для решения его бизнес-задач.
